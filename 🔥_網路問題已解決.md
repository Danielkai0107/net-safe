---

### 方案 2️⃣：使用 mDNS

**適合**：macOS/iOS 開發，不想每次都執行命令

**配置**：
```typescript
// 使用電腦名稱（不會因為換網路而改變）
export const API_BASE_URL = 'http://DanieldeMacBook-Air.local:3001/api';
```

**優點**：
- ✅ 設定一次，永久有效
- ✅ 切換網路無需修改
- ✅ 完全無感

**設定步驟**：
1. 編輯 `apps/mobile/src/config.local.ts`
2. 改為：`http://DanieldeMacBook-Air.local:3001/api`
3. 編輯 `apps/admin/.env`
4. 改為：`VITE_API_URL=http://DanieldeMacBook-Air.local:3001/api`
5. 重啟前端，完成！

---

### 方案 3️⃣：部署到雲端（推薦）🚀

**適合**：徹底解決，任何網路都能用

**最簡單的方式 - Railway（5 分鐘）**：

```bash
# 1. 安裝 CLI
npm install -g @railway/cli

# 2. 部署
cd apps/backend
railway login
railway init
railway up

# 3. 獲取 URL（例如：https://your-app.railway.app）

# 4. 更新前端配置
# apps/admin/.env
VITE_API_URL=https://your-app.railway.app/api

# apps/mobile/src/config.local.ts
export const API_BASE_URL = 'https://your-app.railway.app/api';
```

**優點**：
- ✅ 一勞永逸
- ✅ 任何網路都能訪問
- ✅ 團隊成員也能使用
- ✅ 穩定可靠

**費用**：每月 $5 免費額度

詳細步驟：查看 `DEPLOY_GUIDE.md`
